## Key Features

- **Code Generation:** Maximo Syntax can generate code using natural language.
- **Inline Autocomplete:** Get intelligent code completions as you type, powered by AI.
- **Task Automation:** Maximo Syntax can automate repetitive coding tasks to save time.
- **Automated Refactoring:** Maximo Syntax can refactor and improve existing code efficiently.
- **MCP Server Marketplace:** Maximo Syntax can easily find and use MCP servers to extend the agent capabilities.
- **Multi Mode:** Plan with Architect, Code with Coder, and Debug with Debugger, and make your own custom modes.
- **Native Maximo AI Integration:** Built-in support for Maximo AI models for optimal performance and intelligence.

## Get Started
